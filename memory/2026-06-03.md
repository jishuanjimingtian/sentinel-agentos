## coderev v1.0.18 — GitHub App 自动审查 + Product Hunt 准备

### GitHub App 自动审查 (`coderev serve`)

**核心文件**：`src/github-app.js` (18KB)

功能：
- 启动 HTTP webhook 服务器，接收 GitHub App 的 pull_request 事件
- JWT 认证（生成 App JWT → 换取 installation token）
- 支持三种审查模式：
  - `comment` — PR review comment（Markdown 汇总）
  - `inline` — 行内 review comments（解析 diff 文件映射）
  - `check` — commit status（pending → success/failure/neutral）
- 可选 auto-approve（无问题的 PR 自动 approve）
- Webhook 签名验证（HMAC-SHA256）
- 跳过 draft PR 和 bot PR
- `/health` 端点用于监控

**部署文档**：`docs/github-app.md`（含创建 App、权限设置、Railway/Docker/PM2/ngrok 部署指南）

**配置**：环境变量或 `.coderevrc.json` 的 `githubApp` 段
